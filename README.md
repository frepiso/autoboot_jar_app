## BOOTABLE JAR APP

## Synopsis

Start a Tomcat and JBoss

## Use

```
> mvn archetype:generate -DgroupId=com.odigeo.myapp -DartifactId=myapp -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

> cd myapp

> mvn clean install

> java -jar target\myapp-1.0-SNAPSHOT.jar
```
